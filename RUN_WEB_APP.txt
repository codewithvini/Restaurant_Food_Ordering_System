🚀 HOW TO RUN THE WEB APP (3 STEPS)

Step 1: Install Flask (30 seconds)
───────────────────────────────────
pip install -r requirements.txt

This installs:
  - Flask (web framework)
  - pymongo (MongoDB)
  - dnspython (required)


Step 2: Setup MongoDB Atlas (5-10 minutes)
───────────────────────────────────────────
⚠️  IMPORTANT: You MUST do this first!

1. Go to: https://mongodb.com/cloud/atlas
2. Sign up (free)
3. Create FREE cluster (M0)
4. Create database user
5. Whitelist IP (0.0.0.0/0 for testing)
6. Get connection string
7. Update config.py with YOUR connection string

📖 Detailed guide: MONGODB_ATLAS_SETUP.md


Step 3: Run the Application (instant!)
───────────────────────────────────────
python app.py


Step 4: Open Your Browser
──────────────────────────
Go to: http://localhost:5000

🎉 Your website is LIVE!

🔧 TROUBLESHOOTING

Issue: "Port already in use"
Solution:
  lsof -i :5000
  kill -9 <PID>

Issue: "MongoDB connection failed"
Solution:
  - Check config.py
  - Verify MongoDB Atlas setup
  - Check internet connection
  - Read MONGODB_ATLAS_SETUP.md

Issue: "Module not found"
Solution:
  pip install -r requirements.txt

Issue: "Template not found"
Solution:
  cd /home/yash/Restaurant_Food_Ordering_System
  ls templates/  # Should show HTML files
  python app.py
